# Bundled fonts

`Inter-Regular.ttf` (weight 400) and `Inter-Bold.ttf` (weight 700) are the
**Inter** typeface by Rasmus Andersson, distributed under the
**SIL Open Font License, Version 1.1** (https://opensource.org/license/ofl-1-1)
— compatible with this repo's MIT license. Source: Google Fonts (`fonts.gstatic.com`,
Inter v20). These are used only server-side by the OG-image renderer (satori);
they are never shipped to the browser.
