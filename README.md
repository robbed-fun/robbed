# ROBBED_

**Launch, trade, and graduate memecoins on Robinhood Chain** 🚀 a pump.fun-style launchpad where sells can never be paused, graduation to Uniswap v3 is permissionless, and LP principal is permanently locked (trading fees claimable by treasury).

**Live:** [robbed.fun](https://robbed.fun) · **Testnet:** [testnet.robbed.fun](https://testnet.robbed.fun)

Under the hood: one-transaction token creation on Robinhood Chain (Arbitrum Orbit L2, chain ID 4663), bonding-curve trading with soft-confirmed (~100ms block) UX and explicit confirmation tiers (soft-confirmed → posted-to-L1 → finalized), and graduation into a Uniswap V3 full-range position.
